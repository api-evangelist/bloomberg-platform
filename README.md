# Bloomberg Platform

The Bloomberg Platform is the integrated technology infrastructure underpinning all Bloomberg professional products and services. It encompasses the data distribution network, cloud and on-premises deployment options, API connectivity layer, and enterprise integration capabilities.

## APIs

- **Bloomberg Open API (BLPAPI)** - Foundational API for Bloomberg data access
- **Bloomberg Cloud Connect** - Cloud-native Bloomberg data connectivity
- **Bloomberg Identity and Access Management** - Authentication and entitlement services
